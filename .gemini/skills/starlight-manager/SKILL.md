# Starlight Manager

## Overview
This skill provides a structured approach to managing a Starlight documentation site, focusing on content creation, navigation management, and configuration updates.

## Quick Start
1. **Add a page**: `node ./scripts/add_page.cjs "Page Title" "folder/path"`
2. **Update sidebar**: Edit `astro.config.mjs` under `starlight.sidebar`.
3. **Customize UI**: Add components to `src/components` and use Starlight's `overrides` or `components` configuration.

## Task Categories

### 1. Content Management
- **Add Markdown/MDX**: Files go in `src/content/docs/`.
- **Frontmatter**: Ensure `title` and `description` are present.
- **Assets**: Store images in `src/assets/` and import them in MDX.

### 2. Navigation & Sidebar
- Managed in `astro.config.mjs`.
- Use `label` and `link` (relative to `src/content/docs/`).
- Support nested groups and autogenerated links.

### 3. Configuration & Styling
- **Global Styles**: Edit `src/styles/custom.css`.
- **Config**: `astro.config.mjs` is the central hub for Starlight options.
