นักวิจัยมักใช้จำนวนต้นต่อกอที่แตกต่างกันไปในการทดลอง ยกเว้นในกรณีที่ต้องการวัดความสามารถในการแตกกอ ซึ่งจำเป็นต้องใช้เพียงหนึ่งต้นต่อหนึ่งกอ อย่างไรก็ตาม มักมีคำถามที่พบบ่อยคือ:

1.  การใช้จำนวนต้นต่อกอที่แตกต่างกันส่งผลเสียต่อการทดลองหรือไม่?

2.  หากต้องใช้จำนวนต้นต่อกอเท่ากันตลอดการทดลอง จำนวนที่เหมาะสมที่สุดคือเท่าใด?

### ผลกระทบของการใช้จำนวนต้นต่อกอที่ไม่คงที่

ผลผลิตเมล็ดข้าวและองค์ประกอบผลผลิตหลายประการจะแปรผันตามจำนวนต้นต่อกอ ดังนั้น การใช้จำนวนต้นต่อกอที่ไม่เท่ากันภายในพล็อตทดลองเดียวกันจะทำให้เกิดความแปรปรวนระหว่างกอเพิ่มมากขึ้น ส่งผลให้ต้องใช้ขนาดตัวอย่างที่ใหญ่ขึ้นในการวัดลักษณะต่าง ๆ ของข้าวเพื่อให้ได้ค่าที่แม่นยำ

| 2 ต้น | 1.45 | 14.1 | 29.2 | 72 |

| 4 ต้น | 2.10 | 20.3 | 28.8 | 69 |
| 5 ต้น | 2.15 | 20.8 | 28.0 | 67 |

**คำอธิบาย:** ตารางนี้แสดงให้เห็นว่าผลผลิตและจำนวนรวงต่อกอมีการเปลี่ยนแปลงตามจำนวนต้นที่ปลูก แต่เมื่อปลูกมากกว่า 3 ต้นต่อกอ ผลผลิตเพิ่มขึ้นไม่มากและจำนวนเมล็ดต่อรวงลดลงเนื่องจากการแข่งขันภายในกอ

### ตารางที่ 11: อัตราการเกิดกอหายตามจำนวนต้นต่อกอ

| จำนวนต้นต่อกอ | กอหาย (%) | ต้นตาย (%) | กอหาย + ต้นตาย (%) | กอที่มีชีวิตและครบถ้วน (%) |
|-------------------|-------------|-------------|---------------------|---------------------------|
| 1 ต้น | 14.2 | 5.1 | 19.3 | 80.7 |
| 2 ต้น | 5.8 | 2.9 | 8.7 | 91.3 |
| 3 ต้น | 3.2 | 1.8 | 5.0 | 95.0 |
| 4 ต้น | 2.1 | 1.2 | 3.3 | 96.7 |
| 5 ต้น | 1.5 | 0.9 | 2.4 | 97.6 |

**คำอธิบาย:** ตารางนี้ชี้ให้เห็นว่าการปลูก 1 ต้นต่อกอ มีอัตราการเกิดกอหายเฉลี่ยสูงถึง 19.3% ในขณะที่การปลูก 2-4 ต้นต่อกอ จะช่วยลดปัญหานี้ลงได้อย่างมาก

### จำนวนต้นกล้าต่อกอที่แนะนำ

#### 1. หลีกเลี่ยงการปลูก 1 ต้นต่อกอ

<Aside type="caution">
หากไม่ได้มีจุดประสงค์เพื่อวัดการแตกกอโดยเฉพาะ ควรหลีกเลี่ยงการปลูกเพียงต้นเดียว เนื่องจากมีโอกาสสูงมากที่จะเกิด "กอหาย" หรือต้นข้าวตาย (อ้างอิงตารางที่ 11)
</Aside>

**เหตุผลที่ควรหลีกเลี่ยง:**
- ความเสี่ยงสูงต่อการเกิดกอหาย (19.3%)
- ความแปรปรวนระหว่างกอสูงกว่า
- ต้องใช้ตัวอย่างขนาดใหญ่ขึ้นในการวัด
- ผลผลิตเฉลี่ยต่อกอต่ำกว่า

#### 2. จำนวนที่เหมาะสม

โดยทั่วไปแนะนำให้ใช้ต้นกล้า **2 ถึง 4 ต้นต่อกอ**

<Aside type="tip">
การเลือกจำนวนต้นต่อกอควรพิจารณาตามวัตถุประสงค์ของการทดลอง ประเภทของพันธุ์ข้าว และสภาพแวดล้อม
</Aside>

**คำแนะนำตามวัตถุประสงค์:**

| วัตถุประสงค์ | จำนวนต้นที่แนะนำ | คำอธิบาย |
|-------------|---------------------|-------------|
| วัดการแตกกอ | 1 ต้น | จำเป็นต้องใช้เพื่อวัดความสามารถในการแตกกออย่างแม่นยำ |
| ทดสอบพันธุ์ | 2-3 ต้น | ช่วยลดความแปรปรวนและกอหาย |
| ทดสอบปุ๋ย | 3-4 ต้น | แข่งขันน้อยช่วยแสดงผลของปุ๋ยชัดเจน |
| ทดสอบโรค/ศัตรูพืช | 2-3 ต้น | ช่วยรักษาความสม่ำเสมอในพล็อต |
| การทดลองในนาเกษตรกร | 3-4 ต้น | ลดความเสี่ยงจากการบริหารจัดการที่ต่างกัน |

#### 3. ความสม่ำเสมอ

ควรใช้จำนวนต้นกล้าต่อกอให้เท่ากันสม่ำเสมอตลอดการทดลองทั้งหมด

<Aside type="note">
หากจำเป็นต้องใช้จำนวนต้นที่แตกต่างกันในบางกรณี ให้บันทึกข้อมูลอย่างละเอียดและพิจารณาใช้จำนวนต้นเป็นตัวแปรอิสระ (covariate) ในการวิเคราะห์
</Aside>

## ความแตกต่างตามพันธุ์ข้าว

พันธุ์ข้าวต่าง ๆ มีความเหมาะสมต่อจำนวนต้นต่อกอที่แตกต่างกัน:

| พันธุ์ข้าว | ลักษณะเฉพาะ | จำนวนต้นที่แนะนำ | คำอธิบาย |
|-------------|------------------|---------------------|-------------|
| ข้าวหอมมะลิ 105 | แตกกอดี ต้นอ่อนแอ | 2-3 ต้น | ต้องการแข่งขันน้อยเนื่องจากแตกกอดี |
| เจ๊กกอว่าน | แข็งแรง ทนทาน | 3-4 ต้น | สามารถรองรับการแข่งขันภายในกอ |
| ขาวดอกสีชมพู 90 | แตกกอปานกลาง | 2-3 ต้น | ช่วยลดกอหายและเพิ่มความมั่นคง |
| ข้าวนาหว่าน | แข็งแรง แตกกอน้อย | 2-3 ต้น | ช่วยเพิ่มจำนวนหน่อและลดกอหาย |
| พันธุ์แป้งหวาน | ต้นเล็ก แตกกอมาก | 2 ต้น | การแข่งขันมากอาจลดผลผลิต |

## การจัดการตามสภาพแวดล้อม

### นาหว่าน (Rainfed Rice)

- **จำนวนต้น:** 3-4 ต้นต่อกอ
- **เหตุผล:** น้ำขาดแคลน ต้นเล็กจะช่วยลดการแข่งขันเพื่อความอยู่รอด

### นาฝังชัน (Irrigated Rice)

- **จำนวนต้น:** 2-3 ต้นต่อกอ
- **เหตุผล:** น้ำเพียงพอ ความแข่งขันน้อย สามารถใช้จำนวนต้นน้อยกว่า

### นาไร่เดียว (Single Cropping)

- **จำนวนต้น:** 2-3 ต้นต่อกอ
- **เหตุผล:** มีเวลาเพียงพอให้กอเจริญเติบโต

### นาไร่สอง (Double Cropping)

- **จำนวนต้น:** 3-4 ต้นต่อกอ
- **เหตุผล:** เวลาเจริญเติบโตสั้น ต้องการต้นมากขึ้น

## การบันทึกและการวิเคราะห์ข้อมูล

### การบันทึกจำนวนต้น

**สิ่งที่ควรบันทึก:**

```
พล็อต: [หมายเลข]
วันที่ปักดำ: [วัน/เดือน/ปี]
จำนวนต้นต่อกอที่วางแผน: [ระบุ]
จำนวนต้นจริงต่อกอที่ปัก: [ระบุ]
ความสมบูรณ์ของต้นกล้า: [ดี/ปานกลาง/อ่อนแอ]
การบันทึกกอที่ปักผิด: [ระบุ]
```

### การวิเคราะห์ข้อมูล

หากใช้จำนวนต้นที่ไม่เท่ากันในการทดลอง สามารถพิจารณาใช้จำนวนต้นเป็นตัวแปรอิสระ (covariate) ในการวิเคราะห์ ANCOVA:

```r
# ตัวอย่างการวิเคราะห์ด้วย R
library(agricolae)

# ข้อมูล
yield <- c(2.5, 2.8, 3.1, 2.9, 3.2)
treatment <- c("T1", "T2", "T3", "T1", "T2")
block <- c(1, 1, 1, 2, 2)
seedlings <- c(2, 3, 2, 3, 2)

# สร้าง data frame
data <- data.frame(yield, treatment, block, seedlings)

# วิเคราะห์ด้วย ANCOVA
model <- aov(yield ~ treatment + block + seedlings, data = data)
summary(model)
```

## แนวทางปฏิบัติจากกรมวิชาการเกษตร

1. **ใช้จำนวนต้นเท่ากัน:** หากเป็นไปได้ ใช้จำนวนต้นเท่ากันตลอดการทดลอง

2. **บันทึกจำนวนต้นอย่างละเอียด:** บันทึกจำนวนต้นจริงที่ปักในแต่ละกอ

3. **ตรวจสอบความถูกต้อง:** ตรวจสอบจำนวนต้นที่ปักหลังจากการปักดำ 1-2 วัน

4. **ใช้ต้นกล้าที่คล้ายกัน:** ใช้ต้นกล้าที่มีความสมบูรณ์ใกล้เคียงกัน

5. **พิจารณาจำนวนต้นเป็น covariate:** หากจำเป็นต้องใช้จำนวนต้นที่แตกต่างกัน
