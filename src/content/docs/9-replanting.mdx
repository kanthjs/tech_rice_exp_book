เพื่อลดเปอร์เซ็นต์ของกอที่หายไปในแปลงทดลองข้าว โดยทั่วไปจะใช้วิธีการปลูกซ่อมกอที่ตาย อย่างไรก็ตาม มักจะมีคำถามเกิดขึ้นเสมอว่า: กอที่ปลูกซ่อมจะเจริญเติบโตเป็นปรกติหรือไม่? ควรทำการปลูกซ่อมไปจนถึงระยะเวลาใด? และควรใช้ต้นกล้าประเภทใดในการปลูกซ่อม?

## ผลของการปลูกซ่อม

ไม่ว่าจะทำการปลูกซ่อมเมื่อใดหรือใช้ต้นกล้าประเภทใด **กอที่ปลูกซ่อมจะเจริญเติบโตแตกต่างจากกอปรกติเสมอ** ผลผลิตเมล็ดของกอที่ปลูกซ่อมจะต่ำกว่ากอปรกติ แม้ว่าจะทำการปลูกซ่อมเร็วเพียง 5 วันหลังการปักดำก็ตาม และยิ่งปลูกซ่อมช้าลงเท่าใด ผลผลิตก็จะยิ่งลดลงมากขึ้นเท่านั้น เพื่อผลลัพธ์ที่ดีที่สุด แนะนำให้ใช้ต้นกล้าที่ถอนแบ่ง (thinned) มาจากกอแถวขอบ มากกว่าการใช้ต้นกล้าจากแปลงกล้าเดิมซึ่งจะให้ผลลัพธ์ที่แย่กว่า

### ข้อมูลเชิงปริมาณของผลกระทบ

> **ข้อมูลจากการทดลอง:** จากงานวิจัยที่ IRRI และกรมวิชาการเกษตร พบว่ากอที่ปลูกซ่อมให้ผลผลิตเฉลี่ยต่ำกว่ากอปรกติดังนี้:
- ปลูกซ่อมภายใน 5 วันหลังปักดำ: ลด 5-10%
- ปลูกซ่อม 7-10 วันหลังปักดำ: ลด 15-20%
- ปลูกซ่อม 15-20 วันหลังปักดำ: ลด 25-35%

### ความแตกต่างตามพันธุ์ข้าว

พันธุ์ข้าวต่าง ๆ มีการตอบสนองต่อการปลูกซ่อมที่แตกต่างกัน:

- **ข้าวหอมมะลิ 105:** มีความไวต่อการถอนปลูกใหม่ แนะนำปลูกซ่อมภายใน 7 วัน
- **เจ๊กกอว่าน:** มีความทนทานต่อการถอนปลูกใหม่ดี สามารถยืดเวลาถึง 10-12 วัน
- **ขาวดอกสีชมพู 90:** ควรปลูกซ่อมภายใน 5-7 วันหลังพบกอตาย

## จุดประสงค์ของการปลูกซ่อม

เนื่องจากกอที่ปลูกซ่อมมีการเติบโตที่แตกต่างจากกอปรกติ **จึงไม่สามารถนำกอเหล่านี้มารวมในการเก็บเกี่ยวหรือการวัดลักษณะทางเกษตรใด ๆ ได้** อย่างไรก็ตาม หากทำการปลูกซ่อมอย่างถูกวิธี กอซ่อมจะช่วย "ปกป้องกอที่ล้อมรอบ" ไม่ให้เติบโตผิดปรกติจากการขาดการแข่งขัน (ดูรายละเอียดในบทที่ 8) ดังนั้น การปลูกซ่อมจึงมีประโยชน์หลักคือการลดจำนวนกอที่ต้องถูกตัดทิ้งจากการเก็บเกี่ยวผลผลิตให้เหลือน้อยที่สุด

## วิธีปฏิบัติที่แนะนำสำหรับการปลูกซ่อม

### 1. ระยะเวลาในการปลูกซ่อม

> **ข้อแนะนำ:** กอที่ตายเร็วกว่า 3 สัปดาห์หลังปักดำควรได้รับการปลูกซ่อมทุกกอ เพื่อรักษาการเจริญเติบโตของกอข้างเคียงให้เป็นปรกติ

**ระยะเวลาที่แนะนำตามช่วงเวลา:**

| ช่วงเวลาหลังปักดำ | การจัดการที่แนะนำ | ผลกระทบต่อกอข้างเคียง |
|-------------------|---------------------|----------------------|
| 0-5 วัน | ปลูกซ่อมทันที | ผลกระทบน้อยมาก |
| 6-10 วัน | ปลูกซ่อมโดยเร่ง | ผลกระทบปานกลาง |
| 11-15 วัน | ปลูกซ่อมเพื่อป้องกันกอข้างเคียง | ผลกระทบมาก |
| 16-21 วัน | พิจารณาปลูกซ่อมหากสำคัญ | ผลกระทบมากมาก |
| มากกว่า 21 วัน | ไม่ควรปลูกซ่อม | ควรทำเครื่องหมายและคัดทิ้ง |

### 2. ต้นที่ใช้ปลูกซ่อม

ให้ใช้ต้นกล้าที่ถอนแบ่งมาจากแถวขอบ (border hills) โดยเฉพาะแถวที่ติดกับพื้นที่ว่าง หากมีไม่เพียงพอ ให้ใช้ต้นกล้าที่ทิ้งไว้เป็นมัดๆ บริเวณปลายแถว (มัดละ 10-15 ต้น)

**ลำดับความเหมาะสมของต้นกล้าสำหรับปลูกซ่อม:**

1. **ต้นกล้าถอนแบ่งจากแถวขอบ** (Best): ต้นกล้าเหล่านี้มีอายุใกล้เคียงกับต้นกล้าที่ปลูกในพล็อตและมีความสมบูรณ์คล้ายกัน

2. **ต้นกล้าที่ทิ้งไว้เป็นมัด:** ต้นกล้าเหล่านี้มีความสมบูรณ์ดีแต่อาจมีอายุต่างจากต้นกล้าในพล็อต

3. **ต้นกล้าจากแปลงกล้าเดิม** (Avoid): แม้จะเป็นพันธุ์เดียวกัน แต่มักมีความอ่อนแอจากการถูกถอนและทิ้งไว้

### 3. ขั้นตอนการปลูกซ่อม

**เตรียมการ:**
1. ตรวจสอบกอตายในพล็อตทุก 2-3 วันหลังปักดำ
2. เตรียมต้นกล้าสำหรับปลูกซ่อมแต่เช้าเพื่อรักษาความชื้นที่ราก
3. จัดเตรียมแปลงเพาะกล้าให้มีต้นกล้าสำรองเพียงพอ

**การปลูกซ่อม:**
1. ถอนต้นกล้าออกจากแถวขอบหรือมัดสำรอง ระวังไม่ให้รากช้ำหรือเหี่ยว
2. ปักดำต้นกล้าลงในตำแหน่งกอตายโดยใช้ระยะและจำนวนต้นเหมือนกับกอปรกติ
3. รดน้ำทันทีหลังปักดำเพื่อให้ดินเกาะติดกับราก
4. ใช้ต้นกล้าที่มีความสูงเท่ากับกอปรกติโดยเท่าที่ทำได้

### 4. การบันทึกข้อมูล

**ทำเครื่องหมายกอที่ปลูกซ่อมทุกกอ** และคัดแยกกอเหล่านี้ออกจากการวัดผลผลิตเมล็ด องค์ประกอบผลผลิต และลักษณะทางการเกษตร

**ข้อมูลที่ควรบันทึก:**

```
วันที่ปักดำ: [วัน/เดือน/ปี]
วันที่พบกอตาย: [วัน/เดือน/ปี]
วันที่ปลูกซ่อม: [วัน/เดือน/ปี]
เลขที่กอที่ปลูกซ่อม: [ระบุเลขที่]
แหล่งที่มาของต้นกล้า: [แถวขอบ/มัดสำรอง/แปลงกล้า]
จำนวนต้นที่ปลูก: [ระบุจำนวน]
หมายเหตุ: [ข้อสังเกตเพิ่มเติม]
```

## แนวทางปฏิบัติจากกรมวิชาการเกษตร (DOA)

1. **ใช้สีระบุกอที่ปลูกซ่อม:** ทาหรือใช้เชือกผูกกอที่ปลูกซ่อมด้วยสีเดียวกันทั้งพล็อตเพื่อจดจำง่าย

2. **ตรวจสอบกอที่ปลูกซ่อม:** ตรวจสอบกอที่ปลูกซ่อม 2-3 วันหลังปลูก หากพบกอตายอีกให้ปลูกซ่อมซ้ำหรือทำเครื่องหมายเพื่อคัดทิ้ง

3. **จัดสรรต้นกล้าอย่างเหมาะสม:** เพาะต้นกล้าเผื่อสำหรับการปลูกซ่อมประมาณ 10-15% ของจำนวนที่ต้องการใช้จริง

4. **เว้นทางเดินให้เหมาะสม:** เว้นทางเดินที่เพียงพอให้สามารถเข้าถึงกอที่ปลูกซ่อมได้ง่ายโดยไม่ทำให้กอปรกติได้รับผลกระทบ

## ผลของกอที่หายไปต่อกอข้างเคียง

<Aside type="caution">
หากไม่ปลูกซ่อมกอที่ตาย กอที่อยู่ติดกับจุดว่างจะได้รับประโยชน์จากการขาดการแข่งขัน ทำให้เจริญเติบโตดีกว่าปกติและให้ผลผลิตสูงกว่า ส่งผลให้เกิดความลำเอียง (bias) ในข้อมูล หากไม่จัดการอย่างถูกต้อง
</Aside>

**กอที่ติดกับจุดว่างจะมีการเปลี่ยนแปลงดังนี้:**
- จำนวนรวงต่อกอเพิ่มขึ้น 10-20%
- ความสูงของต้นเพิ่มขึ้น 5-10%
- ผลผลิตเมล็ดเพิ่มขึ้น 15-25%
- อัตราการแตกหน่อเพิ่มขึ้น

ดังนั้น การปลูกซ่อมไม่ได้มีเป้าหมายเพื่อให้กอที่ปลูกซ่อมให้ผลผลิตเท่ากอปรกติ แต่เพื่อป้องกันให้กอข้างเคียงเจริญเติบโตผิดปรกติ
